# StatusView

`StatusView` displays a message independent status for "Web" and "Mobile App".

<Preview>
  <Story name="default">
    <PublishingStatus.StatusView />
  </Story>
</Preview>

`as` and `children` prop override defaults.  
`children` controls only text and _not_ platform statuses.

## churchCenterAppIsActive

When true, `churchCenterAppIsActive` results in a green check by "Mobile App".

<Preview>
  <Story name="churchCenterAppIsActive">
    <PublishingStatus.StatusView churchCenterAppIsActive />
  </Story>
</Preview>

## churchCenterWebIsActive

When true, `churchCenterWebIsActive` results in a green check by "Web".

<Preview>
  <Story name="churchCenterWebIsActive">
    <PublishingStatus.StatusView churchCenterWebIsActive />
  </Story>
</Preview>

## Custom Properties

CSS Custom Properties control the color of platform statuses:

- `--publishing-status--product-enabled--color`
- `--publishing-status--product-not-enabled--color`

<Preview>
  <Story name="custom properties">
    <PublishingStatus.StatusView
      style={{
        "--publishing-status--product-enabled--color": "olivedrab",
        "--publishing-status--product-not-enabled--color": "tomato",
      }}
      churchCenterWebIsActive
    />
  </Story>
</Preview>
