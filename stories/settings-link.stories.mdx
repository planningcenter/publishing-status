# SettingsLink

`SettingsLink` is a an anchor that gets `href` from `ViewController` context.

href value is on context for two reasons:

- The URL is shared between views and menu button items
- The value is nulified if `churchCenterWebIsActive` is **falsy**

## Defaults

`SettingsLink` with `ViewController`

<Preview>
  <Story name="default">
    <PublishingStatus.ViewController userIsOrgAdmin settingsURL="#">
      <PublishingStatus.SettingsLink />
    </PublishingStatus.ViewController>
  </Story>
</Preview>

## `children`

`children` are used to customize messaging.

<Preview>
  <Story name="children">
    <PublishingStatus.ViewController userIsOrgAdmin settingsURL="#">

        Seriously.{" "}
        <strong>
          Update <em>your</em> settings
        </strong>
        .
      </PublishingStatus.SettingsLink>
    </PublishingStatus.ViewController>
  </Story>
</Preview>
