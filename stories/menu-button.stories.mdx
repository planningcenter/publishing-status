# MenuButton

`MenuButton` is a button for use as the disclosure button for the publishing status dropdown.

It includes a down chevron and warning icon to indicate when nothing is published.

## `as`

`as` is used to integrate `MenuButton` into any dropdown library.

This example uses [Reach UI](https://reach.tech/menu-button) — [recommended by Global Design](https://planningcenter.design/react-recommended).

```jsx
<Reach.Menu>
  <PublishingStatus.ViewController>
    <PublishingStatus.MenuButton as={Reach.MenuButton} />
  </PublishingStatus.ViewController>
</Reach.Menu>
```
